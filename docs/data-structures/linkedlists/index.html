Linked List

What is linkedlist ?


Linked List is a sequence of data structures, which are connected together via links. Each struct consists of two parts, data part which stores the relevant data and link/pointer that stores the refrence to the next data block. Linked list is second most used data strucutre after array.
You might be thinking if it is just list of data, we already have arrays for that so what is the point of using a linked list, Right?
Well one disadvantage of using arrays to store data is that arrays are static strcutures and therefore cannot be easily extended or reduced to fit the data set. Arrays are also expensive to maintain new insertions and deletions.
Second major disadvantage is that, memory allocation of array is uniform, i.e. knowing address of any one of the memory block, you can get address of any other memory block of the array. This property helps you to get the required data from an array in O(1) but restricts the max size of an array to an order of O(106). Whereas memory allocation in linkedlist is dynamic, this means that each data block has a randomly allocated memory location. Due to this property of linkedlist, there are no maximum size constraints. This is very useful when you are working with large amount of data that has to be handled.
