Stacks

What are Stacks ?


A stack is an linear data structure, commonly used in most of the programming languages. It follow a paticular order in which operations are performed. It allows adding and removing elements in a particular order. Every time a element is added it goes on the top of the stack, and only element that can be removed is the element that is at the top of the stack, just like a pile of objects.
Most common occurence of stack in programming is during the concept of recursion. Recursion uses stack to store data at every step. Thats why in an unclosed recursion, we get stack overflow error.

Features

  • Stack is ordered list
  • It follows LIFO (Last In First Out) or FILO (First In Last Out) order
  • push() function is used to insert a new element in the stack
  • pop() fuction is used to remove an element from the stack
  • Both insertion and removal are allowed at only one end of Stack called Top
  • Both insertion and removal are of order O(1) and serching is of order O(n)
  • Stack is said to be overflow state when it is completely full

            struct node

            // Adding an element to the stack
            void Push(struct node **head,int key)
            {
            	struct node *temp;
            	temp = malloc(sizeof(struct node));
            	temp->data = key;
            	if((*head) == NULL)
            		(*head) = temp;
            	else
            	{
            		temp->next = (*head);
            		(*head) = temp;
            	}
            }

            // Deleting an element from the stack
            void Pop(struct node **head)
            {
            	(*head) = (*head)->next;
            }

            struct node *last(struct node *head)

            	while(head->next != NULL)
            		head = head->next;
